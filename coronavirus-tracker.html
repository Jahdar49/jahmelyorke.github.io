About Coronavirus-Tracker

Coronavirus-Tracker is a tracker application that utilizes the Spring-Boot framework and maven plugin. The application makes a get request to a csv file url which contains the latest number of Coronavirus cases in various states and countries around the world. The csv file is then processed through the Spring life cycle, parsed, and rendered in an html format using the thymleaf dependency. The tracker is updated on a daily basis and even tracks the spike in cases from the previous day.
